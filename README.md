# Freerooms by CSESOC

A web application designed to aid UNSW students in finding vacant CAT rooms.

## Running on your local machine

### v2 Frontend

##### Project setup

```
npm install
```

##### Compiles and hot-reloads for development

```
npm run serve
```

##### Compiles and minifies for production

```
npm run build
```

### v2 Backend

##### Project setup

```
npm install
```

#### Run with npm

```
npm start
```
